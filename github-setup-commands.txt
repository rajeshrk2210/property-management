First Time Git & Github Setup
==============================
Steps Followed
1. git init
2. git add .
3. git commit -m "any message"
4. git remote add origin git@github.com:rajeshrk2210/property-management.git
5. git remote set-url origin https://rajeshrk2210:<TOKEN>@github.com/rajeshrk2210/property-management.git
6. git push --set-upstream origin master
